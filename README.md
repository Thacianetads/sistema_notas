# Sistema de Classificação de Alunos em C

Este é um programa simples desenvolvido em linguagem C para gerenciar a situação acadêmica de uma turma. O usuário insere o número de alunos, seus nomes e notas, e o sistema classifica cada um entre Aprovado, Recuperação ou Reprovado.
🚀 Funcionalidades:
- Leitura da quantidade total de alunos.
- Classificação automática baseada nos critérios:
  - nota >= 7: Aprovado .
  - nota >= 5 e < 7: Recuperação.
  - Nota < 5: Reprovado.
Exibição de um resumo ao final da execução.
